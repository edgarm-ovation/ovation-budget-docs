# Later Features — deferred from the 8-week build

These features are **intentionally not built during the 8-week H0 demo** (see [roadmap-8-weeks.md](../../ovation-platform-docs/product/roadmap-8-weeks.md)). Each was cut because it's a large time-sink that doesn't prove the core value — *replacing Excel for budget calculation*. The specs are preserved here so the work continues in a future phase rather than being lost.

| # | Feature | Why deferred | Target |
|---|---------|--------------|--------|
| 01 | [Authentication & Roles](./01-authentication.md) | Real Entra ID SSO + full RBAC is heavy. A demo role/user switcher (🔵 stretch) covers H0. | Phase 2 |
| 04 | [File Import (Excel/CSV)](./04-file-import.md) | Highest-risk feature in the plan. Data is **seeded** instead for the demo. | Phase 2 |
| 09 | [Excel Export](./09-excel-export.md) | ClosedXML plumbing that proves no core value. Browser-print of the approval view (🔵 stretch) covers any demo need. | Phase 2 |

For the broader future-phase backlog (benchmarking, historical migration, reporting, GL integration, subcontractor portal, etc.), see [../deferred.md](../deferred.md).
