# Ovation Budget Platform - 8 Week Demo Mockup

Fresh OpenAI mockup created from the confirmed 8-week feature scope and the JSON seed data in `ovation-platform-docs/jsons`.

This version intentionally uses the same visual structure as `Prototypes-UI/ovation-platform-mockup Open Ai/index.html`: Microsoft sign-in, Ovation green/cream palette, top navigation, dashboard project cards, panel tables, and project workspace sidebar.

## Open

Open `index.html` in a browser. It is a standalone static prototype with no install, server, login, or internet required.

## Why This Folder Exists

The existing reference mockup at `Prototypes-UI/ovation-platform-mockup Open Ai/index.html` was not changed. This folder is a new presentation-ready copy focused on what can be built in the next 8 weeks.

## What It Shows

- Seeded project dashboard, led by West Henderson Apartments L3.
- Budget level navigation from L0 to L3.
- Editable budget table mock behavior with save and audit trail.
- Bid leveling with real bidder names and values from `proposal_bids.json`.
- Markup engine view based on `markups.json`.
- L2 vs L3 variance summary using benchmark/baseline-style data.
- Submit, approve, reject, lock, and SHA-256 snapshot demo behavior.
- In-app notification bell and feed.
- Ordered 8-week delivery scope with deferred items separated.
